
# Recursion Practice Questions – JavaScript

## Question 1: Flatten a Nested Array

**Problem Statement:**  
Write a recursive function that takes a nested array of any depth and returns a flat array with all values.

**Expected Input:**  
An array that may contain numbers or other arrays, e.g.  
`[1, [2, [3, 4], 5], 6]`

**Expected Output:**  
A flat array with all values in order, e.g.  
`[1, 2, 3, 4, 5, 6]`


---

## Question 2: Palindrome Check (Recursive)

**Problem Statement:**  
Write a recursive function to check if a given string is a palindrome. A palindrome reads the same forward and backward.

**Expected Input:**  
A lowercase single-word string with no spaces or special characters, e.g.  
`"level"`

**Expected Output:**  
A boolean indicating whether the input is a palindrome, e.g.  
`true` or `false`
